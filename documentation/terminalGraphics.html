terminalGraphics Class Documentation

Description

The terminalGraphics class is designed to allow Python developers to draw text graphics onto the terminal, as well as providing the abillity to draw basic shapes and lines to simplify graphics creatoin. Currently only works for Windows based systems, Unix support will be implemented at a later date.

screen

Is a list containing one internal list for each vertical row on the terminal screen, and within each list, contains a string consisting of only a space, " ", with each position representing each horizontal column on the terminal. This creates a screen[y][x] list to acess any (x,y) "pixel" on the terminal. Origin is (0,0), in the bottom right corner of the screen, and Y max is height - 1, and X max is length - 1.

length

length contains the integer value of the length of the terminal screen in characters. Directly coresponds to len(screen[0]), or the length of any of the internal lists contained witiin screen.

height

height contains the integer value of the height of the terminal screen in characters. Directly coresponds to len(screen).

screenPosition

screenPosition is a string which refers to the position of the position of the screen relative to the terminal window. Currently, the only supported options are "center" and "full."
"center" will draw the screen in the coordinate center of the terminal screen.
"full" will draw the screen to fill up the entire terminal.

Methods

terminalGraphics([forcedSize, screenPosition])

Return Value: instance of the terminalGrphics class
Parameters:
forcedSize:
Default: False
Refers to whether size of of terminal should be set to a certain size. To set this value pass in a list containg ints for the length and height of the screen. Ex. [lengthInt, heightInt]
screenPosition:
Default: "full"
String that determines where the screen will be drawn in relation to the terminal. See screenPosition

newScreen()

Return Value: new and empty instance of the screen variable
Parameters: None

clearScreen()

Return Value: None
Parameters: None
Clears the screen from all characters.

refresh()

Return Value: None
Parameters: None
Draws the current screen stored in memory onto the terminal.

drawPoint(point, [symbol])

Return Value: None
Parameters:
point: [x,y] list of where to draw a point on the screen.
symbol: String of a single char to draw at specifed point. Default value is "V".

drawVerticalLine(x, yPoint, [symbol])

Return Value: None
Parameters:
x: x position of where vertical line should be placed.
yPoint: [y0, y1] list of y points where line should start from and end at.
symbol: String of char used to draw line, default symbol is "|".

drawLine(point1, point2, [symbol])

Return Value: None
Parameters:
point1: [x,y] position of where line should begin.
point2: [x,y] position of where line should end.
symbol: String of char used to draw line, default symbol is "/".
