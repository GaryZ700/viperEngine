Viper Engine Documentation

Viper Engine is a terminal based game engine built in python to allow developers to create python based games right from their terminal. Future iterations of Viper will expand to creating graphic based games that utilize the GPU and potentially even as an engine capable of creating 3D games.

viper([height, length, screenPosition, FPS, layers])

Return Value: Instance of the viper class.
Parameters:
height: Integer representing the height of the terminal screen in characters. Default value is 50 chars.
length: Integer representing the length of the terminal screen in characters. Default value is 70 chars.
screenPosition: String that specifies where screen should be drawn in relation to the terminal. Default value is "center".
FPS: Integer that signifies frames per second, default value is 1.
layers: Integer of how many animation layers should exist on top of base layer 0. Default value is 1.

update()

Return Value: None
Parameters: None
Update function "ticks" viper and instructs the game engine to progress one cycle. Should be run from an endless while loop in the main game script.
