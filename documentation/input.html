Global Variables

keyBinders

keyBinders is a dictonary contaning sublists with keys refering to keyboard keys to listen to. The associated values to each key is a list, with the first index containg an input event keyword, and the second position containing either a reference to a function to execute, or false.
Ex. {"a":["onKeyUp",functionReference]}
Input Events: "onKeyUp", "onKeyDown", or "whileKeyDown"
Funtion: Reference to function to execute, or False if no function is to be executed.

lastKeyUp

String of last key that the user released.

lastKeyDown

String of last key that was held down.

lastKeyDown

String of key that is currently being held down.

Global Methods

addKeyBinding(key, event, function)

Return Value: None
Parameters:
key: string of character to bind event to.
event: string of keyboard event to listen to. See keyBinders for more information.
function: Reference to function to execute upon occurance of event, default value is False, no function should be executed upon completion of the event.
If key binding alreadly exists, existing key binding is overwritten.

updateInput()

Return Value: None
Parameters: None
Checks keyboard input and updates lastKeyDown, lastKeyUp, and currentKeyDown. Also checks and executes any functions in keyBinders.
